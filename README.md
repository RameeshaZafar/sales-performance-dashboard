\# Sales Performance Dashboard



Interactive Power BI dashboard analyzing 10,000+ sales transactions to track revenue growth, profit margins, and regional performance.



\## Tools Used

\- \*\*Python\*\* (Pandas, NumPy, Matplotlib, Seaborn) — Data cleaning \& EDA

\- \*\*SQL\*\* — Business insights queries

\- \*\*Power BI\*\* — Interactive dashboard

\- \*\*Git/GitHub\*\* — Version control



\## Key Insights

\- Total Revenue: $2.30M

\- Total Profit: $286K

\- Top 20% products generate majority of revenue (Pareto analysis)

\- West region leads in sales performance
